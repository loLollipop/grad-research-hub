Design system: 研途 Hub

- Query: 理工科研究生科研工作台 低疲劳 高密度 日常行动
- Matched profile: dashboard
- Style direction: data-dense
- Palette: warm-paper + slate-sage
- Typography: Geist + Noto Sans SC + JetBrains-style mono fallback
- Effects: compact panels, soft paper layers, precise separators
- Landing/layout bias: summary row + filters + charts
- Avoid: heavy admin tables, saturated blue blocks, 3D charts, crowded legends, uneven spacing
- Notes: 首屏先回答“今天先做什么”，长列表优先提供三条行动队列；配置与低频字段隐藏到设置中心或弹窗。
