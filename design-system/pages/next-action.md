# Page override: next-action

- Base authority: ../MASTER.md
- Record only page-specific deviations here.

## Purpose

- 首页必须先回答“今天先做什么”，再给用户进入文献、实验、成果和笔记的路径。
- 不把首页做成统计大屏；统计只作为行动解释，不抢占第一层注意力。

## Rhythm Band

- 使用“开工前 / 深度工作 / 收口前 / 组会前”的一天节奏，而不是抽象模块名。
- 每个节点只放一个短值和一个行动理由，保证忙碌用户能扫一眼就继续工作。
- 视觉上使用低饱和纸面背景、细连接线、小图标节点，避免深色大块造成疲劳。

## Interaction

- 首页行动入口应继续复用已有页面和笔记生成能力，不新增配置或新的维护负担。
- hover 只增强边框和节点颜色，不做会造成布局跳动的放大动效。
